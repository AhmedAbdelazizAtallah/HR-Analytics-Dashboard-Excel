# 📊 HR Analytics Dashboard (Microsoft Excel)

An interactive Human Resources Analytics Dashboard built in Microsoft Excel to analyze employee data, salaries, hiring trends, departments, and workforce distribution.

---

## 📌 Project Overview

This dashboard helps HR managers and business stakeholders monitor workforce metrics through interactive visualizations.

The project was developed using Microsoft Excel with Pivot Tables, Pivot Charts, Slicers, Timeline, and Map Charts.

---

## 🎯 Objectives

- Analyze employee demographics
- Monitor salary distribution
- Track hiring trends over the years
- Compare departments and work types
- Analyze payment methods
- Explore employee distribution by country

---

## 📈 Dashboard KPIs

- Total Employees
- Total Salaries
- Average Experience (Years)

---

## 📊 Dashboard Features

- Total Salary

- Tax Calculation

- Salary Transfer Methods
